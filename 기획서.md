## 1. 서비스 개요

### 1.1 서비스명 (가칭)
**FeedbackFlow** — AI 기반 성과 리뷰 & 피드백 어시스턴트

### 1.2 미션
> "개략적인 내용만 작성하면, 목표에 맞는 고품질 리뷰와 피드백이 완성된다."

리더와 팀원 모두가 피드백 작성에 소요하는 시간과 부담을 줄이면서도, 회사의 비전/미션/핵심가치와 연계된 **구체적이고 실행 가능한** 리뷰·피드백을 생성하는 AI 서비스.

### 1.3 핵심 가치 제안 (Value Proposition)
| 대상 | 기존 Pain Point | FeedbackFlow 해결 |
|------|----------------|-------------------|
| **팀원** | 셀프리뷰 작성이 막연하고, 목표와 연결짓기 어려움 | 개략적 키워드/문장만 입력하면 KPT 프레임워크에 맞춰 목표 연계 리뷰 자동 생성 |
| **리더** | 팀원마다 톤·깊이가 다른 피드백 작성에 과도한 시간 소요 | 셀프리뷰 기반으로 상황별 톤(경고/지원/칭찬)에 맞는 피드백 초안 자동 생성 |
| **HR/경영진** | 피드백 품질 편차, 핵심가치와 괴리 | 회사 핵심가치·조직 목표 자동 연계로 일관된 피드백 품질 확보 |

## 2. 타겟 시장

### 2.1 타겟 고객
- **조직 규모**: 5~50인 스타트업·중소기업
- **현재 상황**: 레몬베이스, FLEX, 원티드스페이스 등 HR 솔루션을 사용 중이거나 구글시트/노션으로 성과관리를 하는 조직
- **핵심 페르소나**:
  - **리더(팀장)**: 3~15명의 팀원을 관리하며 분기별 1on1과 연말 종합 피드백 수행
  - **팀원**: 분기마다 셀프리뷰를 작성해야 하는 실무자
  - **HR 담당자**: 피드백 프로세스를 설계·운영하는 관리자

### 2.2 시장 포지셔닝
```
[HR 솔루션(레몬베이스, FLEX)]  ←→  [FeedbackFlow]  ←→  [범용 AI(ChatGPT)]
       프로세스 관리 중심              리뷰/피드백 콘텐츠 생성 특화        맥락 없는 범용 생성
```
- HR 솔루션과 **경쟁이 아닌 보완 관계** — 어떤 HR 툴을 쓰든 리뷰/피드백 "내용 작성"을 도와주는 AI 레이어

## 3. 핵심 기능

### 3.1 전체 서비스 플로우

```
┌─────────────────────────────────────────────────────────────┐
│                    [초기 설정 (온보딩)]                        │
│  회사 비전/미션 → 핵심가치 → 당해연도 조직 목표 입력             │
└─────────────────────┬───────────────────────────────────────┘
                      │
                      ▼
┌─────────────────────────────────────────────────────────────┐
│              [STEP 1] 목표 설정 & 합의 (연초)                  │
│  팀원: 성과목표 + 성장목표 초안 작성                             │
│  AI: 핵심가치/조직목표와 연계하여 목표 고도화 제안                 │
│  리더: 검토 후 합의·확정                                       │
└─────────────────────┬───────────────────────────────────────┘
                      │
                      ▼
┌─────────────────────────────────────────────────────────────┐
│           [STEP 2] 분기별 셀프리뷰 작성 (팀원)                  │
│  팀원: 개략적 내용(키워드, 간단 문장) 입력                       │
│  AI: KPT(Keep/Problem/Try) 프레임워크로 정리                   │
│      + 합의된 성과/성장 목표와 자동 매핑                         │
│      + 핵심가치 연계 포인트 제시                                │
│  팀원: 검토·수정 후 제출                                       │
└─────────────────────┬───────────────────────────────────────┘
                      │
                      ▼
┌─────────────────────────────────────────────────────────────┐
│         [STEP 3] 리더 피드백 생성 (분기별 1on1)                 │
│  입력: 셀프리뷰 + 리더의 개략적 코멘트/평가                      │
│  AI: 상황 자동 판단 → 톤 매칭                                  │
│      🔴 경고: 단호하고 명확한 피드백                             │
│      🟡 지원: 친절하지만 명확한 가이드                           │
│      🟢 칭찬: 구체적 성과 인정 + 다음 스텝 제시                  │
│  리더: 검토·수정 후 전달                                       │
└─────────────────────┬───────────────────────────────────────┘
                      │
                      ▼
┌─────────────────────────────────────────────────────────────┐
│          [STEP 4] 연말 종합 피드백 (연 1회)                     │
│  AI: 4개 분기 데이터 종합 분석                                  │
│      → 성장 궤적 시각화                                        │
│      → 연간 종합 피드백 초안 생성                               │
│      → 차기연도 목표 제안                                      │
└─────────────────────────────────────────────────────────────┘
```

### 3.2 기능 상세

#### F1. 온보딩 — 회사 컨텍스트 설정
| 항목 | 설명 |
|------|------|
| 회사 비전/미션 | 텍스트 입력 또는 이미지 업로드(OCR 지원) |
| 핵심가치 | 항목별 입력 (명칭 + 설명 + 행동지표) |
| 당해연도 조직 목표 | 팀/부서별 목표 입력 |
| 피드백 가이드라인 | 회사 고유 피드백 원칙 설정 (선택) |

#### F2. 목표 설정 어시스턴트
- **입력**: 팀원이 작성한 성과/성장 목표 초안
- **AI 처리**:
  - 조직 목표와의 정합성 체크
  - SMART 기준으로 구체화 제안
  - 핵심가치와 연결되는 행동지표 추천
- **출력**: 고도화된 목표안 (팀원·리더가 함께 검토)

#### F3. 셀프리뷰 작성 어시스턴트
- **입력**: 개략적 내용 (자유 형식)
  - 예: "Q1에 신규 캠페인 3건 런칭함. 데이터 분석 역량 부족 느낌. 다음엔 GA4 공부하고 싶음"
- **AI 처리**:
  - KPT 프레임워크로 자동 구조화
  - 합의된 목표 대비 달성도 매핑
  - 핵심가치 실천 사례 연결
  - 구체적 수치/사례 보강 가이드
- **출력 예시**:
  ```
  [Keep - 유지할 점]
  • 신규 캠페인 3건 성공 런칭 (성과목표 1 연계)
    → 핵심가치 '인텐트 데이터로 마케팅 통합을 리딩하자' 실천
    → 캠페인별 구체적 성과 지표를 추가하면 더 강력한 리뷰가 됩니다

  [Problem - 개선할 점]
  • 데이터 분석 역량 강화 필요 (성장목표 2 연계)
    → 구체적으로 어떤 상황에서 부족함을 느꼈는지 사례를 추가해보세요

  [Try - 시도할 점]
  • GA4 학습 계획 수립 (성장목표 2 연계)
    → 핵심가치 '지속 성장과 공유의 문화' 연결
    → 학습 후 팀 내 공유 세션 계획을 추가하면 좋습니다
  ```

#### F4. 리더 피드백 생성기
- **입력**: 팀원 셀프리뷰 + 리더의 개략적 평가/코멘트
  - 예: "캠페인은 잘했는데 데이터 역량이 심각하게 부족. 다음 분기에 개선 안 되면 역할 조정 필요"
- **AI 처리**:
  - 상황 심각도 자동 판단
  - 톤 매칭 (아래 3단계)
  - 핵심가치/목표 기반 구체적 피드백 구성
  - 실행 가능한 액션 아이템 제시

**톤 분류 체계:**

| 톤 | 트리거 조건 | 피드백 스타일 |
|----|-----------|-------------|
| 🔴 **경고** | "심각", "반복", "역할조정", "경고" 등 | 단호하고 직접적. 현재 상황의 심각성을 명확히 전달. 구체적 개선 기한과 기대 수준 제시. |
| 🟡 **지원/코칭** | "부족", "성장필요", "도움", "고민" 등 | 친절하지만 명확. 현재 수준을 인정하되 구체적 성장 방향 제시. 리소스/지원 방안 포함. |
| 🟢 **칭찬/격려** | "잘했", "초과달성", "성장", "기여" 등 | 구체적 성과 인정. 조직에 미친 영향 명시. 다음 레벨 도전 과제 제안. |

- **출력 예시 (경고 톤)**:
  ```
  [피드백 - Q1 성과 리뷰]

  캠페인 런칭 성과에 대해 먼저 인정합니다. 3건의 캠페인을
  계획대로 실행한 것은 '인텐트 기반 마케팅 통합' 가치에
  부합하는 좋은 실행력이었습니다.

  다만, 데이터 분석 역량에 대해 솔직하게 이야기해야 합니다.
  이번 분기 성장목표였던 '데이터 기반 의사결정 역량 강화'에서
  유의미한 진전이 보이지 않았습니다. 이는 단순 스킬 이슈가 아니라,
  우리 팀의 핵심가치인 '의도 탐구에서 시작하자'를 실천하기 위한
  기본 역량과 직결됩니다.

  [명확한 기대사항]
  • Q2 중 GA4 기초~중급 과정 이수 완료 (6월 말까지)
  • 7월 캠페인부터 직접 데이터 분석 리포트 작성
  • 월 1회 진행 상황 공유 (별도 체크인)

  Q2 종료 시점에 개선이 확인되지 않을 경우,
  역할 범위 조정을 검토할 수 있습니다.
  함께 방법을 찾겠지만, 주도적인 노력이 반드시 필요합니다.
  ```

#### F5. 연말 종합 피드백
- 4개 분기 셀프리뷰 + 리더 피드백 종합 분석
- 성과목표 달성률 요약
- 성장 궤적 서술 (어디서 → 어디까지 왔는지)
- 핵심가치 실천도 종합 평가
- 차기연도 목표 제안

## 4. 사용자 플로우 (User Journey)

### 4.1 HR/관리자 플로우
```
가입 → 회사정보 입력 → 핵심가치 설정 → 조직목표 입력
→ 팀 구성(리더/팀원 초대) → 리뷰 주기 설정
```

### 4.2 팀원 플로우
```
초대 수락 → 성과/성장 목표 초안 작성 → [AI 고도화] → 리더와 합의
→ (분기마다) 개략적 내용 입력 → [AI KPT 정리] → 검토/수정 → 제출
```

### 4.3 리더 플로우
```
팀원 목표 검토/합의 → (분기마다) 팀원 셀프리뷰 확인
→ 개략적 코멘트 입력 → [AI 피드백 생성] → 톤 선택/조정
→ 검토/수정 → 1on1 진행 → 전달
```

## 5. 기술 아키텍처

### 5.1 시스템 구성도
```
┌──────────────┐     ┌──────────────────┐     ┌─────────────────┐
│   Frontend   │────▶│   Backend API    │────▶│   Database      │
│  (Next.js)   │◀────│   (Node.js)      │◀────│  (PostgreSQL)   │
└──────────────┘     └────────┬─────────┘     └─────────────────┘
                              │
                              ▼
                     ┌──────────────────┐
                     │   AI Engine      │
                     │  (Claude API)    │
                     │                  │
                     │ • 프롬프트 관리    │
                     │ • 톤 분류 엔진    │
                     │ • 컨텍스트 관리    │
                     └──────────────────┘
```

### 5.2 기술 스택 (권장)
| 레이어 | 기술 | 이유 |
|--------|------|------|
| Frontend | Next.js + TypeScript | SSR, 빠른 개발, 풍부한 에코시스템 |
| Backend | Node.js (Express/Fastify) | 프론트엔드와 언어 통일, 빠른 프로토타이핑 |
| Database | PostgreSQL + Prisma ORM | 관계형 데이터에 적합, 타입 안전성 |
| AI | Claude API (Anthropic) | 한국어 품질 우수, 긴 컨텍스트 윈도우, 톤 제어 우수 |
| 인증 | NextAuth.js | 소셜 로그인 + 이메일 인증 |
| 배포 | Vercel + AWS RDS | 빠른 배포, 확장성 |
| 파일저장 | AWS S3 | 핵심가치 이미지 등 |

### 5.3 AI 프롬프트 설계 핵심
```
[System Prompt 구조]

1. 회사 컨텍스트 주입
   - 비전/미션
   - 핵심가치 (항목별 상세)
   - 당해연도 조직 목표

2. 개인 컨텍스트 주입
   - 합의된 성과/성장 목표
   - 이전 분기 리뷰/피드백 이력

3. 작업 지시
   - 역할 (셀프리뷰 어시스턴트 / 리더 피드백 생성기)
   - 출력 프레임워크 (KPT / 피드백 구조)
   - 톤 지정 (경고/지원/칭찬)

4. 가드레일
   - 사실 기반만 서술 (추측 금지)
   - 입력에 없는 성과 창작 금지
   - 핵심가치 연결은 자연스럽게 (억지 연결 금지)
```

## 7. MVP 개발 로드맵

### Phase 1 — MVP Core (8주)
> 핵심 가치 검증: "AI가 만든 리뷰/피드백이 실제로 쓸 만한가?"

| 주차 | 마일스톤 | 상세 |
|------|---------|------|
| 1-2주 | 기본 인프라 | 프로젝트 세팅, DB 스키마, 인증, 회사 온보딩 UI |
| 3-4주 | 목표 관리 | 성과/성장 목표 CRUD, 리더-팀원 합의 플로우 |
| 5-6주 | 셀프리뷰 AI | KPT 기반 셀프리뷰 작성 어시스턴트 (핵심 기능) |
| 7-8주 | 리더 피드백 AI | 톤 분류 + 피드백 생성기 (핵심 기능) |

**Phase 1 산출물**: 핵심 플로우 동작하는 웹앱 + 5개사 파일럿

### Phase 2 — 고도화 (4주)
| 기능 | 설명 |
|------|------|
| 연말 종합 피드백 | 4분기 데이터 종합 분석 리포트 |

## 8. 비즈니스 모델

### 8.1 가격 정책 (안)
| 플랜 | 대상 | 가격 | 포함 기능 |
|------|------|------|----------|
| **Free** | 5인 이하 | 무료 | 월 10회 AI 생성, 기본 KPT 리뷰 |
| **Starter** | ~15인 | 월 49,000원 | 무제한 AI 생성, 톤 매칭 피드백, 분기 리뷰 |
| **Growth** | ~50인 | 월 149,000원 | 전체 기능, 연말 종합 피드백, 히스토리, 우선 지원 |
| **Enterprise** | 50인+ | 별도 협의 | 커스텀 연동, 전담 지원, SLA |

### 8.2 핵심 지표 (KPI)
- **활성 조직 수**: 서비스를 사용 중인 회사 수
- **리뷰 생성 수**: AI로 생성된 셀프리뷰/피드백 수
- **수정률**: AI 초안 대비 사용자 수정 비율 (낮을수록 품질 좋음)
- **NPS**: 리더/팀원 만족도
